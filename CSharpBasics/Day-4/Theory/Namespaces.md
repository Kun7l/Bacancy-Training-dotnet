# Namespaces

**Importance of namespaces in large systems**

i ) **Prevent naming conflicts :** namespaces isolate groups having similar named (Class ,method,functions etc.). 

ii ) **Improves organization :** it improves organization of codes. Modules having related features are grouped under same namespace.

iii ) Improves readability & code becomes maintanable 

iv ) **Scalability :** allows systems to add new functionality without breaking the existing code.
