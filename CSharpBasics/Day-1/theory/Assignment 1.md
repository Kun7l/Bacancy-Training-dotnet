1. **How a console application is created**

A console application is a type of program that runs on a terminal/console. It is text based (No GUI).  Uses text based input and outputs. 

To create a Console application Create new project > select **Console App (.NET) > Create**

1. **Purpose of the Main() method**
    
    Main() method tells the runtime where to start the progam execution. it controls the flow of execution
    
2. **Difference between Console.Write() and Console.WriteLine()**
    
    Console.Write() prints line starting with same line and Console.WriteLine() starts with a new line.
